# TreasureDAO Migration to ZKsync - February 2024

## Overview

TreasureDAO has voted to migrate from Arbitrum to Matter Labs' ZKsync, with an overwhelming **99.5% majority** in favor of the move.

## Timeline

### Initial Plan (Pre-February 2024)
- TreasureDAO planned to build **TreasureChain** using **Arbitrum Orbit**
- Games were already deployed on Arbitrum

### February 2024 - Decision Point
- TreasureDAO announced it was open to exploring other options
- Multiple networks reached out:
  - **Optimism**
  - **ZKsync**
  - Others

### Decision Made
- **99.5% vote** in favor of migrating to **ZKsync**
- Migration process initiated

## Migration Details

### Ecosystem Impact
- **10+ ecosystem games** affected by the migration
- Includes both announced and **unannounced titles**
- TreasureDAO working closely with all games for smooth transition

### Technical Implementation

#### Testnet: Topaz

- Allows developers and players to test **ZK Stack technology**
- Provides hands-on experience before mainnet launch
