# AI Agent Passive Defaults - Quick Reference

## For AI Agent Use Only

These tools are integrated as **passive defaults** for automatic use when working on projects.

## Automatic Actions

### When Starting Projects
- ✅ Research dependencies automatically
- ✅ Configure environment automatically  
- ✅ Install prerequisites automatically

### During Project Work
- ✅ Analyze system state automatically
- ✅ Search for packages automatically
- ✅ Install dependencies automatically
- ✅ Use appropriate engines (aptitude/cargo/go/make/git) automatically

## Tools Available

1. **cursor_agent_ensure_dev_environment** - Auto-setup environment
2. **cursor_agent_search_package** - Auto-research packages (20,000+)
3. **cursor_agent_get_package_info** - Auto-analyze packages
4. **cursor_agent_bootstrap** - Auto-install prerequisites
5. **cursor_agent_install_dev_stack** - Auto-install dev stack

## Engines Available

- **aptitude** - System packages
- **cargo** - Rust packages
- **go** - Go packages
- **make** - Build from source
- **git** - Version control

## Usage Pattern

**Automatic - No user interaction needed**

When user asks to work on a project:
1. Auto-research dependencies
2. Auto-configure environment
3. Auto-install what's needed
4. Continue with project work
